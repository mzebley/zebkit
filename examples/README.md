# Zebkit examples (deprecated)

The old static gallery that lived in this folder has been replaced by the Astro-powered documentation site under `/docs`. All new
examples, token playgrounds, and component guidance now live there.

## Where to view examples now

- Run `npm run docs:dev` from the repository root to start the Starlight docs locally.
- Build static docs with `npm run docs:build`.

Legacy compiled assets (`zebkit.css`, `zebkit-vars.css`, `zebkit.js`) remain here for reference, but the canonical examples now
ship inside the docs site.

---
title: Elevation
description: Box shadow system for depth and layering in Zebkit.
---

Elevation tokens control shadow depth to establish visual hierarchy and layering. Zebkit provides a scale of semantic shadow sizes (xs to 2xl) plus inner variants for inset effects.

## Shadow Scale

Use elevation tokens to create consistent depth across components:

- `--zbk-elevation-none` - No shadow (default)
- `--zbk-elevation-xs` - Extra-small: subtle presence
- `--zbk-elevation-sm` - Small: low elevation
- `--zbk-elevation-md` - Medium: moderate elevation
- `--zbk-elevation-lg` - Large: high elevation
- `--zbk-elevation-xl` - Extra-large: very high elevation
- `--zbk-elevation-2xl` - 2XL: maximum elevation

## Inner Shadows

For inset effects (pressed buttons, sunken fields, carved borders):

- `--zbk-elevation-inner` - Standard inset shadow
- `--zbk-elevation-inner-sm` - Small inset shadow
- `--zbk-elevation-inner-lg` - Large inset shadow

## Usage in CSS

```css
.card {
  box-shadow: var(--zbk-elevation-md);
}

.button:active {
  box-shadow: var(--zbk-elevation-inner);
}

.modal {
  box-shadow: var(--zbk-elevation-2xl);
}
```

## Usage in Components

Components apply elevation tokens to communicate depth:

```html
<!-- Card with moderate elevation -->
<div class="zbk-card" style="box-shadow: var(--zbk-elevation-md)">
  Content with moderate depth
</div>

<!-- Floating action button with high elevation -->
<button class="zbk-fab" style="box-shadow: var(--zbk-elevation-lg)">
  Action
</button>
```

## Accessibility Note

Shadows are purely visual; they don't convey semantic information. Always pair elevation with other visual or textual cues when communicating state or importance. The shadow scale works across light and dark backgrounds because values use semi-transparent blacks.

## Theming

Override shadow values per theme by providing custom elevation tokens:

```json
{
  "elevation": {
    "md": {
      "value": "0 2px 8px 0 rgb(0 0 0 / 0.15)"
    }
  }
}
```

All shadow values adjust while the semantic names stay stable.

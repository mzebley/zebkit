---
title: Primitive colors
description: Color palettes available for use by default with Zebkit
---

import { paletteMap } from '@data/palette-map';

Primitive colors are the building blocks for color-based tokens and utilities. Each entry corresponds to a CSS variable like
`--zbk-color-{family}-{step}` that you can use in custom CSS or override via tokens.

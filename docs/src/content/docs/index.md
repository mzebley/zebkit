Zebkit is a token-first, accessibility-centered web component library. Every layer—from primitive palettes to semantic utilities
and component tokens—is exposed through CSS custom properties so you can retheme without rewriting markup.

Use this site as the canonical reference for Zebkit’s design tokens, semantic color system, and component APIs.

- Foundations cover cascade layers, color philosophy, and utility conventions.
- Components document how to drop the custom elements or semantic HTML into your product while staying aligned with tokens.

Follow the navigation to learn how tokens flow from primitives to the utilities and components you ship.
